# Angular Material Demo App

> WIP

> This SPA is intended to use in [cordova](http://ngcordova.com/) platform

### Prerequisites
* node js ^6.9.1
* npm ^3.10.8


### Quick Start
* Clone this repo
* Install dependencies
```
npm install
```
* Start browserSync local server
```
npm start
```
